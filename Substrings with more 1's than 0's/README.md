### 📝 Problem Statement

Given a binary string s consists only of 0s and 1s. Calculate the number of substrings that have more 1s than 0s.
Examples:


### Input:
 s = "011"


### Output:
 4


### Explanation:
 There are 4 substring which has more 1s than 0s. i.e "011","1","11" and "1"



### Input:
 s = "0000"


### Output:
 0


### Explanation:
 There is no substring with more 1s than 0s


### Constraints:
1 < |S| <  6 * 104
