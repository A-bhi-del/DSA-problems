### 📝 Problem Statement

Given an array a[] of n numbers, find the Least Common Multiple (LCM) of all the elements of the array. The LCM of an array is the smallest number that is divisible by every element of the array.
Examples: 


### Input:
 a[] = [1, 2, 8, 3]


### Output:
 24


### Explanation:
 LCM of 1, 2, 8 and 3 is 24 since 24 is the smallest number divisible by all of them.


### Input:
 a[] = [1, 8]


### Output:
 8


### Explanation:
 LCM of 1 and 8 is 8 since 8 is the smallest number divisible by both.



### Constraints:
1 <= n <= 101 <= a[i] <= 20
