### 📝 Problem Statement

Given an integer n, find the sum of the count of divisors for each number from 1 to n.
Examples :


### Input:
 n = 2


### Output:
 3


### Explanation:
1 has 1 divisor: {1}.
2 has 2 divisors: {1, 2}. 
The total count is 1 + 2 = 3.



### Input:
 n = 5


### Output:
 10


### Explanation:
1 has 1 divisor: {1}.
2 has 2 divisors: {1, 2}. 3 has 2 divisors: {1, 3}.4 has 3 divisors: {1, 2, 4}.5 has 2 divisors: {1, 5}.
The total count is 1 + 2 + 2 + 3 + 2 = 10.



### Constraints:
1 ≤ n ≤ 105
